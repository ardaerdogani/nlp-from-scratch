# NLP Project: Text Generation Models and Chatbots

Complete NLP system demonstrating probabilistic modeling, sequential neural networks, and attention-based architectures. All models are implemented from scratch using PyTorch with no pretrained weights.

## Task 1: Text Generation Models

Six models total (3 architectures x 2 levels):

| Model | Character-Level | Word-Level |
|-------|----------------|------------|
| Markov Chain | N-gram transitions (order=5) | N-gram transitions (order=2) |
| RNN | Embedding + RNN + Linear | Embedding + RNN + Linear |
| LSTM | Embedding + LSTM + Linear | Embedding + LSTM + Linear |

### Dataset
- Procedurally generated English text corpus
- 12,000+ words across 10 topics (nature, science, technology, history, etc.)
- Generated by `data/generate_dataset.py`

### Model Details

**Markov Chain**: Stores transition probabilities between N-gram states. Character-level uses order-5 (5 characters of context), word-level uses order-2 (bigrams). Generation uses weighted random sampling.

**RNN**: Standard Elman RNN with embedding layer, 2-layer stacked RNN, and linear output. Trained with cross-entropy loss and Adam optimizer.

**LSTM**: Long Short-Term Memory with embedding layer, 2-layer stacked LSTM, and linear output. Gating mechanism (forget, input, output gates) provides better gradient flow.

### Evaluation
- 10 test prompts per level
- Outputs compared in a table (each output >= 20 characters)
- Training time, memory usage, and loss tracked

## Task 2: Chatbot Models

### Dataset
- 2,000+ QA pairs covering geography, science, math, technology, history, and general knowledge
- Template-based generation with variations
- Generated by `data/generate_qa_dataset.py`

### Model Details

**LSTM Chatbot (Seq2Seq with Attention)**:
- Bidirectional LSTM encoder
- LSTM decoder with Bahdanau attention
- Teacher forcing with decreasing ratio
- Embedding dimension: 128, Hidden dimension: 256

**Transformer Chatbot**:
- Full encoder-decoder Transformer
- Multi-head self-attention (4 heads) implemented from scratch
- Sinusoidal positional encoding
- 3 encoder layers, 3 decoder layers
- d_model: 128, d_ff: 512

### Evaluation
- 10 test questions
- Side-by-side response comparison
- Training loss curves (saved as PNG)
- Qualitative analysis

## Running Individual Components

```bash
# Generate datasets only
python -m data.generate_dataset
python -m data.generate_qa_dataset

# Train text generation models only
python -m train.train_text_gen

# Train chatbot models only
python -m train.train_chatbot
```

## Technical Notes

- All models implemented from scratch in PyTorch
- No pretrained weights or external APIs used
- Training uses gradient clipping (max norm = 1.0)
- Learning rate scheduling applied to all neural models
- Transformer uses warmup + inverse square root decay
- LSTM chatbot uses decreasing teacher forcing ratio
- Dataset generation is deterministic (seeded random)
