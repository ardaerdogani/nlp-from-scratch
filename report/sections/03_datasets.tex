\section{Datasets}
\label{sec:datasets}

Both datasets are generated programmatically from templates and seeded with a
fixed random seed so that every pipeline run produces identical data.

\subsection{Text Corpus (Task 1)}

The corpus is produced by \texttt{task1\_text\_generation/data/generate\_dataset.py}.
It contains \textbf{12,000+ words} of English prose spread across ten topic domains:
science, technology, history, geography, food, sports, arts, medicine, education,
and nature. Sentences are constructed from randomized subject–verb–object templates
to ensure lexical diversity while remaining grammatically plausible.

Key statistics:
\begin{itemize}
    \item Total characters: $\approx$83,000
    \item Vocabulary (unique words): $\approx$1,200
    \item Character vocabulary: 68 unique symbols
\end{itemize}

\subsection{QA Dataset (Task 2)}

The QA dataset is produced by \texttt{task2\_chatbot/data/generate\_qa\_dataset.py}.
It contains \textbf{2,000+ question–answer pairs} across six domains: geography,
science, mathematics, technology, history, and general knowledge. Each pair consists
of a natural-language question (input) and a short factual answer (target).

Key statistics:
\begin{itemize}
    \item Total pairs: 2,000+
    \item Average question length: $\approx$7 words
    \item Average answer length: $\approx$12 words
    \item Shared vocabulary size: $\approx$1,500 tokens
\end{itemize}

Because both datasets are small relative to real-world corpora, model outputs
exhibit limited coherence. This is expected and serves as a useful illustration
of data-hunger in neural language models.
